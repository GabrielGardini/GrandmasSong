Esta Política de Privacidade descreve como o aplicativo Grandmas Song ("nós", "nosso" ou "aplicativo") coleta, usa e protege as informações dos usuários.

1. Informações coletadas
O aplicativo não coleta dados pessoais identificáveis dos usuários.

Podemos coletar informações não pessoais de forma automática, como:

Dados de uso (tempo de uso, interações dentro do app)
Informações do dispositivo (modelo, sistema operacional)
Dados anônimos para análise de desempenho
Essas informações são utilizadas apenas para melhorar a experiência do usuário.

2. Uso das informações
As informações coletadas são utilizadas para:

Melhorar o funcionamento do aplicativo
Corrigir erros e bugs
Analisar o desempenho do app
Aperfeiçoar a experiência do usuário
3. Compartilhamento de dados
Não compartilhamos, vendemos ou alugamos dados pessoais dos usuários.

Podemos utilizar serviços de terceiros (como ferramentas de análise) que coletam informações anônimas para nos ajudar a entender o uso do aplicativo.

4. Armazenamento de dados
Como não coletamos dados pessoais, não armazenamos informações sensíveis dos usuários.

5. Segurança
Adotamos medidas razoáveis para proteger as informações do aplicativo contra acesso não autorizado, alteração ou destruição.

6. Privacidade infantil
O aplicativo é adequado para todas as idades e não coleta intencionalmente dados de crianças.

7. Alterações nesta política
Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que você revise esta página regularmente.
